
# 江苏省生产建设项目水土保持方案·标准化图集分类汇总

## 一、编制依据
1. **国家标准**：《生产建设项目水土保持技术标准》(GB50433-2018)
2. **地方法规**：《江苏省水土保持条例》、《江苏省生产建设项目水土保持技术指南》
3. **地域特征**：南京丘陵岗地/长江漫滩地貌、南方红壤区降雨侵蚀特征、海绵城市建设要求。
