# template.docx 标签植入指南

> 用 Word 打开你的「水土保持方案报告书_完整底板.docx」，按下面的清单逐项替换/插入。
> 语法：`{{变量名}}` = 简单替换，`{%tr for ...%}` = 表格行循环

---

## 一、章节正文区（8处）

在每章的正文区域（标题下方），插入对应标签。撰稿智能体的输出会填充到这里。

| 位置 | 插入内容 |
|------|----------|
| 第1章 综合说明 正文区 | `{{ch1_text}}` |
| 第2章 项目概况 正文区 | `{{ch2_text}}` |
| 第3章 水土保持分析与评价 正文区 | `{{ch3_text}}` |
| 第4章 水土流失预测 正文区 | `{{ch4_text}}` |
| 第5章 水土保持措施 正文区 | `{{ch5_text}}` |
| 第6章 水土保持监测 正文区 | `{{ch6_text}}` |
| 第7章 投资估算及效益分析 正文区 | `{{ch7_text}}` |
| 第8章 方案实施的保障措施 正文区 | `{{ch8_text}}` |

**注意**：正文标签和表格标签共存。表格由 TplCtx 直接渲染，正文由撰稿Agent生成。

## 操作建议

1. 先做 **表7.1-1（投资表）** 和 **表4.3-10（预测表）** — 这两张是审计重点
2. 再做 **表5.3-1（新增措施）** 和 **表3.2-1（已有措施）** — 循环表格
3. 最后补齐其余的标签
4. 全程用 Word 的「查找替换」功能批量操作，不用一个个找
